JavaScript Functions

Learn how to organize code with functions and reuse logic.

What are Functions?

A function is a reusable block of code that performs a specific task. Functions help you organize your code, make it more readable, and avoid repeating the same code multiple times.

Creating Functions

You can declare a function using the function keyword:

Example: Basic Function
// Function declaration function greet() { console.log("Hello, World!"); } // Call the function greet(); // Output: Hello, World! greet(); // Output: Hello, World! // Function with parameters function greetPerson(name) { console.log("Hello, " + name + "!"); } greetPerson("Alice"); // Output: Hello, Alice! greetPerson("Bob"); // Output: Hello, Bob!

Output (in console):

Hello, World!

Hello, World!

Hello, Alice!

Hello, Bob!

Parameters and Return Values

Functions can accept parameters and return values:

Concept Description Example
Parameters Input values for the function function add(a, b)
Return Value Output value from the function return a + b;
Arguments Actual values passed to function add(5, 3)

Functions with Return Values

Example: Functions Returning Values
// Function that returns a value function add(a, b) { return a + b; } function multiply(x, y) { return x * y; } let result1 = add(5, 3); console.log(result1); // Output: 8 let result2 = multiply(4, 6); console.log(result2); // Output: 24 // Use returned values in operations let total = add(10, 20) + multiply(5, 2); console.log(total); // Output: 40 // Function that returns a string function createMessage(name, age) { return name + " is " + age + " years old"; } let message = createMessage("Alice", 25); console.log(message); // Output: Alice is 25 years old

Output (in console):

8

24

40

Alice is 25 years old

Arrow Functions

Arrow functions are a shorter, more modern way to write functions in JavaScript:

Example: Arrow Functions
// Traditional function function square(x) { return x * x; } // Arrow function (equivalent) const squareArrow = (x) => { return x * x; }; // Shortened arrow function const double = (x) => x * 2; console.log(square(5)); // Output: 25 console.log(squareArrow(5)); // Output: 25 console.log(double(5)); // Output: 10 // Arrow function with no parameters const greet = () => "Hello!"; console.log(greet()); // Output: Hello!

Output (in console):

25

25

10

Hello!

Interactive Function Example

Example: Interactive Calculator with Functions
<!DOCTYPE html> <html> <body> <h2>Simple Calculator</h2> <input type="number" id="a" placeholder="Number 1"> <input type="number" id="b" placeholder="Number 2"> <button onclick="showSum()">Add</button> <button onclick="showProduct()">Multiply</button> <p id="output"></p> <script> function add(x, y) { return x + y; } function multiply(x, y) { return x * y; } function showSum() { let num1 = Number(document.getElementById("a").value); let num2 = Number(document.getElementById("b").value); document.getElementById("output").innerHTML = num1 + " + " + num2 + " = " + add(num1, num2); } function showProduct() { let num1 = Number(document.getElementById("a").value); let num2 = Number(document.getElementById("b").value); document.getElementById("output").innerHTML = num1 + " × " + num2 + " = " + multiply(num1, num2); } </script> </body> </html>

Output:

Simple Calculator

Enter numbers and click a button to see results

Key Takeaways

  • Functions are reusable blocks of code that perform specific tasks
  • Functions can accept parameters and return values
  • Arrow functions provide a shorter syntax for writing functions
  • Use functions to organize and simplify your code
  • Functions make code more maintainable and reusable
