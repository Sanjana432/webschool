JavaScript & the DOM

Learn how to interact with and manipulate HTML elements using JavaScript.

What is the DOM?

The DOM (Document Object Model) is a programming interface that represents the HTML document as a tree of objects. It allows JavaScript to dynamically access and modify the content, structure, and style of HTML pages.

Accessing Elements

There are several ways to access HTML elements from JavaScript:

Method Purpose Example
getElementById() Get element by ID getElementById("myId")
querySelector() Get first element matching selector querySelector(".myClass")
querySelectorAll() Get all elements matching selector querySelectorAll("p")
getElementsByClassName() Get elements by class name getElementsByClassName("myClass")

Accessing and Modifying Content

Example: Getting and Setting Element Content
<!DOCTYPE html> <html> <body> <h1 id="title">Original Title</h1> <p class="text">This is a paragraph</p> <button onclick="changeContent()">Change Content</button> <script> function changeContent() { // Get element by ID document.getElementById("title").innerHTML = "New Title!"; // Get first element with class document.querySelector(".text").innerHTML = "This paragraph has been changed!"; // Get text content (without HTML tags) let text = document.querySelector(".text").textContent; console.log(text); } </script> </body> </html>

Modifying Styles

You can change an element's style dynamically using JavaScript:

Example: Changing Styles
<!DOCTYPE html> <html> <body> <div id="box">Click to change color</div> <button onclick="changeStyle()">Change Style</button> <script> function changeStyle() { let box = document.getElementById("box"); // Change multiple styles box.style.backgroundColor = "#667eea"; box.style.color = "white"; box.style.padding = "20px"; box.style.fontSize = "18px"; box.style.borderRadius = "8px"; box.style.textAlign = "center"; } </script> </body> </html>

Output:

Click to change color

Event Listeners

Event listeners allow you to respond to user actions like clicks, typing, and mouse movements:

Example: Using Event Listeners
<!DOCTYPE html> <html> <body> <button id="myButton">Click Me</button> <input id="myInput" placeholder="Type something"> <p id="output"></p> <script> // Click event document.getElementById("myButton").addEventListener("click", function() { document.getElementById("output").innerHTML = "Button was clicked!"; }); // Input event document.getElementById("myInput").addEventListener("input", function() { document.getElementById("output").innerHTML = "You typed: " + this.value; }); </script> </body> </html>

Type or click to see output

Creating and Removing Elements

Example: Creating and Removing Elements
<!DOCTYPE html> <html> <body> <div id="list"></div> <button onclick="addItem()">Add Item</button> <button onclick="removeItem()">Remove Item</button> <script> let count = 0; function addItem() { count++; let item = document.createElement("div"); item.innerHTML = "Item " + count; item.id = "item" + count; item.style.padding = "10px"; item.style.margin = "5px 0"; item.style.background = "#f5f5f5"; document.getElementById("list").appendChild(item); } function removeItem() { let item = document.getElementById("item" + count); if (item) { item.remove(); count--; } } </script> </body> </html>

Output:

Click the buttons to add or remove items from the list above

Key Takeaways

  • The DOM represents HTML as a tree of objects
  • Use methods like getElementById() and querySelector() to access elements
  • You can modify element content with innerHTML and textContent
  • Use style property to change element styles dynamically
  • addEventListener() allows you to respond to user interactions
  • You can create and remove elements dynamically with JavaScript
