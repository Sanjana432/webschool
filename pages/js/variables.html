JavaScript Variables & Data Types

Learn how to store and work with data in JavaScript.

What are Variables?

Variables are containers for storing data values. You use variables to store information that you want to use later in your program. Think of a variable like a labeled box where you can store different types of information.

Declaring Variables

In JavaScript, you can declare variables using let, const, or var:

Keyword Purpose Use Case
let Block-scoped variable (preferred) Most modern JavaScript code
const Block-scoped constant (cannot be changed) Values that shouldn't change
var Function-scoped variable (older style) Legacy code (avoid in new code)

Using Variables

Example: Declaring and Using Variables
// Using let to declare a variable let name = "Alice"; let age = 30; let score = 95.5; // Using const for values that don't change const PI = 3.14159; const country = "USA"; // Display variables console.log(name); // Output: Alice console.log(age); // Output: 30 console.log("Score: " + score); // Output: Score: 95.5 // Change variable value age = 31; console.log(age); // Output: 31 // Try to change const (this will cause an error) // PI = 3.14; // Error: Assignment to constant variable

Output (in console):

Alice

30

Score: 95.5

31

Data Types

JavaScript has several data types you can work with:

Type Description Example
String Text enclosed in quotes "Hello" or 'World'
Number Integers or decimals 42 or 3.14
Boolean true or false true or false
Array List of values [1, 2, 3]
Object Collection of key-value pairs {name: "John"}

Working with Different Data Types

Example: Different Data Types
// String let firstName = "John"; let lastName = "Doe"; let fullName = firstName + " " + lastName; console.log(fullName); // Output: John Doe // Number let x = 10; let y = 20; let sum = x + y; console.log(sum); // Output: 30 // Boolean let isStudent = true; let isTeacher = false; console.log(isStudent); // Output: true // Array let fruits = ["apple", "banana", "orange"]; console.log(fruits[0]); // Output: apple console.log(fruits.length); // Output: 3 // Object let person = { name: "Alice", age: 25, city: "New York" }; console.log(person.name); // Output: Alice

Output (in console):

John Doe

30

true

apple

3

Alice

Key Takeaways

  • Variables are containers for storing data
  • Use let and const for declaring variables (prefer const for constants)
  • JavaScript has several data types: string, number, boolean, array, object
  • You can change variable values after declaring them (except const)
  • Variables make your code more flexible and reusable
