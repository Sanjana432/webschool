JavaScript Introduction

Learn the basics of JavaScript and how to add interactivity to web pages.

What is JavaScript?

JavaScript is a programming language that runs in the web browser. It allows you to add interactivity to web pages - like responding to user clicks, validating form data, animating elements, and much more.

If HTML is the skeleton and CSS is the skin, JavaScript is the brain and movement of your web page.

Three Ways to Add JavaScript

There are three main ways to include JavaScript in your HTML:

Method Location Use Case
Inline JavaScript In HTML attributes like onclick Quick interactions on one element
Internal JavaScript In <script> tag in HTML JavaScript for one page
External JavaScript In separate .js file JavaScript for entire website (recommended)

Your First JavaScript Program

Example: Hello World with JavaScript
<!DOCTYPE html> <html> <head> <title>My First JavaScript</title> </head> <body> <h1>JavaScript Example</h1> <script> console.log("Hello, World!"); document.write("Hello from JavaScript!"); </script> </body> </html>

Output:

JavaScript Example

Hello from JavaScript!

(Check browser console for "Hello, World!")

JavaScript Syntax Basics

JavaScript code is organized into statements and blocks:

Example: Basic JavaScript Syntax
// This is a comment - single line /* This is a multi-line comment */ // Statements end with semicolons var name = "John"; var age = 25; // Logs to console console.log("Name: " + name); console.log("Age: " + age); // Mathematical operations var sum = 10 + 5; var product = 10 * 5; console.log(sum); // Output: 15 console.log(product); // Output: 50

Output (in console):

Name: John

Age: 25

15

50

Interacting with HTML

JavaScript can interact with HTML elements and change the page dynamically:

Example: Changing HTML Content
<!DOCTYPE html> <html> <body> <h1 id="title">Original Title</h1> <p id="message">Original message</p> <button onclick="changeContent()">Click Me</button> <script> function changeContent() { document.getElementById("title").innerHTML = "Title Changed!"; document.getElementById("message").innerHTML = "Message changed with JavaScript!"; } </script> </body> </html>

Output (before click):

Original Title

Original message

Output (after click):

Title Changed!

Message changed with JavaScript!

Common Output Methods

There are several ways to output data in JavaScript:

Method Purpose Example
console.log() Output to console console.log("text")
document.write() Write to HTML page document.write("text")
innerHTML Change element content element.innerHTML = "text"
alert() Show popup message alert("message")

Key Takeaways

  • JavaScript adds interactivity and dynamic behavior to web pages
  • JavaScript can be added inline, internal, or external
  • JavaScript statements end with semicolons
  • JavaScript can interact with HTML elements
  • The DOM (Document Object Model) allows you to access and modify HTML
  • There are multiple ways to output data in JavaScript
