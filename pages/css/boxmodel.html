The CSS Box Model

Understand how every element on a web page is a box with margins, borders, padding, and content.

What is the Box Model?

The CSS box model describes the boxes which wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

Understanding the box model is fundamental to controlling spacing and layout on web pages.

Box Model Components

From inside to outside, the box model consists of:

Component Purpose Example
Content The actual text, images, or other content Content text here
Padding Space inside the border, around content padding: 20px;
Border Line around padding border: 2px solid black;
Margin Space outside the border margin: 10px;

Visualizing the Box Model

Example: Box Model Visualization
.box { content: "Hello"; padding: 20px; border: 3px solid blue; margin: 30px; }

Output:

margin: 30px
border: 3px solid blue
Content
padding: 20px

Padding

Padding creates space inside an element, between the content and the border.

Example: Padding Property
/* All sides */ .box1 { padding: 20px; } /* Vertical | Horizontal */ .box2 { padding: 10px 20px; } /* Top | Right | Bottom | Left */ .box3 { padding: 10px 15px 20px 5px; } /* Individual sides */ .box4 { padding-top: 10px; padding-right: 20px; padding-bottom: 30px; padding-left: 15px; }

Output:

20px padding on all sides

Content with equal padding

10px vertical, 20px horizontal

Content with different padding

Margin

Margin creates space outside an element, outside the border. It creates distance between this element and others.

Example: Margin Property
/* All sides */ .box1 { margin: 20px; } /* Vertical | Horizontal */ .box2 { margin: 10px 20px; } /* Center element horizontally */ .box3 { width: 200px; margin: 0 auto; }

Output:

Element with 20px margin

Notice the space around this element

Another element

There's space between the elements

Centered element

Border

The border property adds a border around an element.

Example: Border Properties
/* Shorthand */ .box1 { border: 2px solid black; } /* Individual sides */ .box2 { border-top: 5px solid red; border-right: 3px solid green; border-bottom: 5px solid blue; border-left: 3px solid orange; } /* Rounded corners */ .box3 { border: 2px solid gray; border-radius: 10px; }

Output:

Solid black border
Different colored borders on each side
Rounded corners with border-radius

Complete Box Model Example

Example: Complete Box Model Demonstration
<!DOCTYPE html> <html> <head> <style> .card { width: 300px; background: white; margin: 20px auto; padding: 20px; border: 3px solid #667eea; border-radius: 12px; } .card h2 { margin-top: 0; color: #333; } .card p { margin: 10px 0; color: #666; line-height: 1.6; } </style> </head> <body> <div class="card"> <h2>Beautiful Card</h2> <p>This demonstrates the complete box model with margin, border, and padding working together.</p> </div> </body> </html>

Output:

Beautiful Card

This demonstrates the complete box model with margin, border, and padding working together.

Key Takeaways

  • Every element is a box with content, padding, border, and margin
  • Padding is inside the border, margin is outside
  • Padding and margin can be set on all sides or individually
  • Borders provide visual boundaries for elements
  • Understanding the box model is essential for layout control
