Responsive Design

Build layouts that adapt beautifully across mobile, tablet, and desktop.

Core Concepts

  • Fluid units: %, vw, rem
  • Flexible media with max-width: 100%
  • Breakpoints using @media
  • Mobile-first styling strategy

Media Query Example

Responsive card grid
.grid { display:grid; grid-template-columns: repeat(3, 1fr); gap:1rem; } @media (max-width: 900px) { .grid { grid-template-columns: repeat(2, 1fr);} } @media (max-width: 600px) { .grid { grid-template-columns: 1fr;} }
