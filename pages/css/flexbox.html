CSS Flexbox

Master the Flexible Box Layout for responsive, modern web design.

What is Flexbox?

Flexbox is a powerful CSS layout tool that makes it easy to design flexible responsive layout structures. It provides an efficient way to arrange, distribute, and align items in a container, even when their size is unknown or dynamic.

Flex Container & Items

To use Flexbox, you need a flex container and flex items inside it:

Example: Creating a Flex Container
<style> .container { display: flex; } .item { flex: 1; } </style> <div class="container"> <div class="item">Item 1</div> <div class="item">Item 2</div> <div class="item">Item 3</div> </div>

Output:

Item 1
Item 2
Item 3

Flex Direction

The flex-direction property determines the direction of flex items in the flex container:

Property Effect
flex-direction: row Items in a horizontal row (default)
flex-direction: column Items in a vertical column
flex-direction: row-reverse Items in a reversed horizontal row
flex-direction: column-reverse Items in a reversed vertical column
Example: Flex Direction
<style> .row { display: flex; flex-direction: row; } .column { display: flex; flex-direction: column; } </style>

Output:

Row Direction:

1
2
3

Column Direction:

1
2
3

Justify Content & Align Items

These properties control how flex items are distributed in the container:

Property Effect
justify-content: flex-start Items at the start (default)
justify-content: center Items centered horizontally
justify-content: space-between Equal space between items
align-items: center Items centered vertically
Example: Justify Content & Align Items
<style> .centered { display: flex; justify-content: center; align-items: center; height: 200px; } .spaced { display: flex; justify-content: space-between; } </style>

Output:

Centered (both horizontally and vertically):

Centered Content

Space Between:

Flexbox Card Layout Example

Here's a practical example of using Flexbox to create a responsive card layout:

Example: Card Layout with Flexbox
<!DOCTYPE html> <html> <head> <style> .container { display: flex; flex-wrap: wrap; gap: 20px; justify-content: center; } .card { width: 250px; background: white; border: 1px solid #ddd; border-radius: 8px; overflow: hidden; box-shadow: 0 2px 8px rgba(0,0,0,0.1); } .card-image { width: 100%; height: 150px; background: linear-gradient(135deg, #667eea, #764ba2); } .card-content { padding: 20px; } .card-content h3 { margin-top: 0; color: #333; } </style> </head> <body> <div class="container"> <div class="card"> <div class="card-image"></div> <div class="card-content"> <h3>Card 1</h3> <p>Description here</p> </div> </div> <div class="card"> <div class="card-image"></div> <div class="card-content"> <h3>Card 2</h3> <p>Description here</p> </div> </div> <div class="card"> <div class="card-image"></div> <div class="card-content"> <h3>Card 3</h3> <p>Description here</p> </div> </div> </div> </body> </html>

Key Takeaways

  • Flexbox is a powerful tool for creating flexible layouts
  • Use display: flex to create a flex container
  • flex-direction controls whether items flow horizontally or vertically
  • justify-content controls horizontal alignment
  • align-items controls vertical alignment
  • Flexbox is essential for creating responsive, modern web designs
