CSS Introduction

Learn how to style HTML elements with CSS.

What is CSS?

CSS stands for Cascading Style Sheets. It is used to style and layout web pages — for example, to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.

If HTML is the skeleton, CSS is the skin, clothes, and makeup of the web page.

Three Ways to Add CSS

There are three main ways to add CSS to your HTML:

Method Location Use Case
Inline CSS Inside HTML elements Quick styling for one element
Internal CSS In <style> tag in <head> Styling for one page
External CSS In separate .css file Styling for entire website (recommended)

CSS Syntax

CSS uses a simple syntax with selectors and properties:

Example: Basic CSS Syntax
/* This is a comment */ h1 { color: blue; font-size: 28px; text-align: center; } p { color: gray; line-height: 1.6; }

Output:

This is a heading

This is a paragraph. Notice the gray color and increased line-height for better readability.

Key Components:

  • Selector - Element to style (h1, p, etc.)
  • Property - What to change (color, size, etc.)
  • Value - What to change it to (blue, 28px, etc.)

CSS Selectors

CSS selectors are patterns used to select the elements you want to style:

Selector Example Selects
Element p { } All <p> elements
Class .blue { } All elements with class="blue"
ID #header { } Element with id="header"
Universal * { } All elements

Using Different Selectors

Example: Different CSS Selectors
<!DOCTYPE html> <html> <head> <style> /* Element selector */ h1 { color: blue; } /* Class selector */ .highlight { background: yellow; } /* ID selector */ #main { margin: 20px; } </style> </head> <body> <h1>Main Title</h1> <p class="highlight">This text is highlighted</p> <div id="main">Main content here</div> </body> </html>

Output:

Main Title

This text is highlighted

Main content here

Common CSS Properties

Here are some of the most commonly used CSS properties:

Property Purpose Example
color Text color color: red;
background-color Background color background-color: blue;
font-size Text size font-size: 20px;
font-family Font type font-family: Arial, sans-serif;
text-align Text alignment text-align: center;
margin Outer spacing margin: 10px;
padding Inner spacing padding: 15px;

Using Common Properties

Example: Common CSS Properties
body { font-family: Arial, sans-serif; background-color: #f5f5f5; } h1 { color: #333; font-size: 32px; text-align: center; } .card { background-color: white; padding: 20px; margin: 10px; border-radius: 8px; }

Output:

Beautiful Styling

This is a styled card with padding, margin, and border-radius.

Key Takeaways

  • CSS is used to style and layout web pages
  • CSS has three main methods: inline, internal, and external
  • CSS uses selectors to target HTML elements
  • CSS properties control appearance like color, size, and spacing
  • External CSS files are the best practice for larger projects
