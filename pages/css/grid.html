CSS Grid

Master CSS Grid for creating two-dimensional layouts and complex designs.

What is CSS Grid?

CSS Grid is a two-dimensional layout system for the web. It lets you lay content out in rows and columns with many unique features that make building complex layouts easier than ever.

While Flexbox is best for one-dimensional layouts, Grid is perfect for two-dimensional layouts.

Creating a Grid Container

To use CSS Grid, you set the display property to grid on a container element:

Example: Basic Grid Layout
<style> .grid-container { display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 20px; } .grid-item { background: #667eea; color: white; padding: 30px; text-align: center; border-radius: 8px; } </style> <div class="grid-container"> <div class="grid-item">Item 1</div> <div class="grid-item">Item 2</div> <div class="grid-item">Item 3</div> </div>

Output:

Item 1
Item 2
Item 3

Grid Template Columns & Rows

Define the structure of your grid with grid-template-columns and grid-template-rows:

Property Purpose Example
grid-template-columns Define column sizes grid-template-columns: 1fr 1fr 1fr;
grid-template-rows Define row sizes grid-template-rows: 100px 200px;
gap Space between items gap: 20px;
repeat() Repeat pattern grid-template-columns: repeat(3, 1fr);

Different Grid Layouts

Here are examples of different grid configurations:

Example: Various Grid Layouts
<!-- 2 equal columns --> .grid-2col { display: grid; grid-template-columns: 1fr 1fr; } <!-- 3 equal columns --> .grid-3col { display: grid; grid-template-columns: 1fr 1fr 1fr; } <!-- Mixed sizes --> .grid-mixed { display: grid; grid-template-columns: 2fr 1fr 1fr; } <!-- Fixed and flexible --> .grid-fixed-flex { display: grid; grid-template-columns: 200px 1fr; }

Responsive Grid Layout

You can use media queries with Grid to create responsive layouts that adapt to different screen sizes:

Example: Responsive Grid
<style> .responsive-grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 20px; } /* Tablet size */ @media (max-width: 768px) { .responsive-grid { grid-template-columns: repeat(2, 1fr); } } /* Mobile size */ @media (max-width: 480px) { .responsive-grid { grid-template-columns: 1fr; } } </style>

Complete Grid Example: Magazine Layout

Here's a practical example of a responsive magazine-style layout using Grid:

Example: Magazine Layout
<!DOCTYPE html> <html> <head> <style> .magazine { display: grid; grid-template-columns: repeat(3, 1fr); gap: 20px; margin: 20px; } .featured { grid-column: span 3; background: linear-gradient(135deg, #667eea, #764ba2); color: white; padding: 40px; border-radius: 8px; } .article-card { background: white; border-radius: 8px; padding: 20px; box-shadow: 0 2px 8px rgba(0,0,0,0.1); } @media (max-width: 768px) { .magazine { grid-template-columns: repeat(2, 1fr); } .featured { grid-column: span 2; } } @media (max-width: 480px) { .magazine { grid-template-columns: 1fr; } .featured { grid-column: span 1; } } </style> </head> <body> <div class="magazine"> <div class="featured"> <h1 style="margin: 0 0 10px 0;">Featured Article</h1> <p>This is the main featured story spanning all columns.</p> </div> <div class="article-card"> <h3>Article 1</h3> <p>Content here</p> </div> <div class="article-card"> <h3>Article 2</h3> <p>Content here</p> </div> <div class="article-card"> <h3>Article 3</h3> <p>Content here</p> </div> </div> </body> </html>

Key Takeaways

  • CSS Grid is perfect for two-dimensional layouts
  • Use display: grid to create a grid container
  • grid-template-columns defines column structure
  • The fr unit represents flexible fractions of space
  • Gap creates space between grid items
  • Use grid-column: span to make items span multiple columns
  • Media queries make grids responsive
