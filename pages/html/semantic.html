Semantic HTML

Use meaningful tags that improve accessibility, SEO, and maintainability.

Why Semantic Tags Matter

Semantic elements describe intent. This makes screen readers, search engines, and teammates understand your page structure quickly.

TagPurpose
<header>Intro area of a page/section
<nav>Primary navigation links
<main>Main page content
<article>Independent piece of content
<section>Thematically grouped content
<footer>Closing content or metadata

Semantic Layout Example

Example
<header>...</header> <nav>...</nav> <main> <article>Blog Post</article> <aside>Related links</aside> </main> <footer>...</footer>
