Images & Links in HTML

Learn how to add images and links to your web pages.

The Image Element

Images are inserted using the <img> tag, which is self-closing. You must specify the image source with the src attribute.

Example: Basic Image
<img src="nature.jpg" alt="Beautiful nature landscape">

Output:

[Image: Beautiful nature landscape]

Image Attributes

Images support several important attributes:

Attribute Purpose
src Path to the image file (required)
alt Alternative text if image can't load or for accessibility
width Image width in pixels
height Image height in pixels
title Tooltip text shown on hover

Using Image Attributes

Here are examples of using different image attributes:

Example: Image with Attributes
<!-- Image with width and height --> <img src="photo.jpg" alt="My photo" width="300" height="200"> <!-- Image with title (shows on hover) --> <img src="sunset.jpg" alt="Beautiful sunset" title="Click to enlarge"> <!-- Linking an image --> <a href="gallery.html"> <img src="thumbnail.jpg" alt="Gallery preview"> </a>

Output:

Image with width and height:

[300x200 image]

Image as a link:

[Clickable thumbnail]

Links in HTML

Links are created using the <a> tag. The href attribute specifies the URL or file to navigate to.

Example: Creating Links
<!-- External link --> <a href="https://example.com">Visit Example</a> <!-- Link to another page on the same site --> <a href="about.html">About Us</a> <!-- Link to a specific section --> <a href="#section-id">Jump to Section</a> <!-- Link that opens in a new tab --> <a href="https://example.com" target="_blank">Open in New Tab</a>

Output:

Visit Example

About Us

Jump to Section

Open in New Tab

Link Attributes

Important attributes for links include:

Attribute Purpose
href URL or file path the link points to (required)
target Where to open the link (_blank for new tab)
title Tooltip text shown on hover
download Download the file instead of navigating

Complete Example

Here's a page combining images and links:

Example: Images and Links Together
<!DOCTYPE html> <html> <body> <h1>My Portfolio</h1> <h2>Project 1</h2> <img src="project1.jpg" alt="Project 1 preview" width="300"> <p><a href="project1.html">View Project</a></p> <h2>Project 2</h2> <img src="project2.jpg" alt="Project 2 preview" width="300"> <p><a href="project2.html">View Project</a></p> <p><a href="https://github.com" target="_blank">Visit My GitHub</a></p> </body> </html>

Output:

My Portfolio

Project 1

[Project 1 preview]

View Project

Project 2

[Project 2 preview]

View Project

Visit My GitHub

Key Takeaways

  • Use <img> to embed images in web pages
  • Always include the alt attribute for accessibility
  • You can control image size with width and height attributes
  • Use <a> tags to create links to other pages or websites
  • Links can open in a new tab using target="_blank"
  • Images can be made clickable by wrapping them in <a> tags
