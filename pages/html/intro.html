HTML Introduction

Learn the basics of HTML and how it structures web pages.

What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used to create web pages. HTML provides the structure and content of websites by using a system of tags and elements.

Think of HTML as the skeleton of a web page. CSS adds the appearance (skin and clothes), and JavaScript adds the behavior (movements and interactions).

HTML Document Structure

Every HTML document follows a basic structure. Here's what a typical HTML page looks like:

Example: Basic HTML Structure
<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>Page Title</title> </head> <body> <h1>Hello, World!</h1> <p>This is my first webpage.</p> </body> </html>

Output:

Hello, World!

This is my first webpage.

Let's break down the parts:

  • <!DOCTYPE html> - Tells the browser this is an HTML5 document
  • <html> - The root element that wraps all content
  • <head> - Contains metadata like title, character encoding, and links to stylesheets
  • <body> - Contains all the visible content of the page

Common HTML Tags

HTML uses various tags to mark different types of content. Here are some of the most common ones:

Tag Purpose Example
<h1> to <h6> Headings (h1 is largest) <h1>Main Title</h1>
<p> Paragraph <p>Text content</p>
<a> Hyperlink <a href="url">Link</a>
<img> Image <img src="image.jpg" alt="Description">
<ul>, <li> Unordered list <ul><li>Item</li></ul>

Your First HTML Page

Now let's create a simple HTML page with multiple elements:

Example: Simple Page with Multiple Elements
<!DOCTYPE html> <html> <head> <title>My First Website</title> </head> <body> <h1>Welcome to My Website</h1> <h2>About Me</h2> <p>I'm learning HTML at Web School!</p> <p>It's a great way to start my journey in web development.</p> <h2>My Interests</h2> <ul> <li>Web Development</li> <li>Learning to Code</li> <li>Building Projects</li> </ul> </body> </html>

Output:

Welcome to My Website

About Me

I'm learning HTML at Web School!

It's a great way to start my journey in web development.

My Interests

  • Web Development
  • Learning to Code
  • Building Projects

Key Takeaways

  • HTML is the foundation of all web pages
  • It uses a system of tags to structure content
  • Every HTML document has a basic structure with <html>, <head>, and <body>
  • Different tags are used for different types of content
  • HTML provides the structure, while CSS and JavaScript add styling and interactivity
