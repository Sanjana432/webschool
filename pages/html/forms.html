HTML Forms

Learn how to create interactive forms to collect user input.

What are Forms?

Forms are used to collect data from users. They include input fields, buttons, checkboxes, radio buttons, and more. The <form> element wraps all form elements together.

Example: Basic Form
<form> <label for="name">Name:</label> <input type="text" id="name" name="name"> <label for="email">Email:</label> <input type="email" id="email" name="email"> <button type="submit">Submit</button> </form>

Input Types

The <input> element supports many different types for various data:

Type Purpose Example
text Single line text input <input type="text">
email Email address input <input type="email">
password Password (masked) <input type="password">
number Numeric input <input type="number">
checkbox Multiple choice selection <input type="checkbox">
radio Single choice selection <input type="radio">

Form Elements

Here are common form elements beyond basic input types:

Example: Various Form Elements
<form> <!-- Text input --> <input type="text" placeholder="Enter text"> <!-- Textarea for multiple lines --> <textarea rows="4"></textarea> <!-- Dropdown select --> <select> <option>Choose an option</option> <option>Option 1</option> <option>Option 2</option> </select> <!-- Checkboxes --> <input type="checkbox"> Option 1 <input type="checkbox"> Option 2 <!-- Radio buttons --> <input type="radio" name="choice"> Choice 1 <input type="radio" name="choice"> Choice 2 <!-- Button --> <button type="submit">Submit</button> </form>

Complete Contact Form Example

Here's a realistic contact form combining multiple form elements:

Example: Contact Form
<form> <h2>Contact Us</h2> <label for="name">Full Name:</label> <input type="text" id="name" required> <label for="email">Email:</label> <input type="email" id="email" required> <label for="subject">Subject:</label> <input type="text" id="subject"> <label for="message">Message:</label> <textarea id="message" rows="5"></textarea> <input type="checkbox"> Subscribe to newsletter <button type="submit">Send Message</button> </form>

Output:

Contact Us

Subscribe to newsletter

Key Takeaways

  • Forms are created with the <form> element
  • Different <input> types collect different kinds of data
  • <textarea> is used for multi-line text input
  • <select> creates dropdown menus
  • Checkboxes allow multiple selections, radio buttons allow only one
  • Always use <label> elements to describe form fields for accessibility
