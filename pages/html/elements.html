HTML Elements & Semantic Markup

Master HTML elements and learn how to use semantic HTML for better structure.

Understanding HTML Elements

An HTML element consists of a start tag, content, and an end tag. Some elements are self-closing and don't have content.

Example: HTML Element Structure
<tagname>Content goes here</tagname> <!-- Examples --> <h1>This is a heading</h1> <p>This is a paragraph</p> <img src="image.jpg" alt="An image"> <!-- Self-closing --> <br> <!-- Line break -->

Output:

This is a heading

This is a paragraph

Image would appear here

Line break appears above

Text Formatting Elements

HTML provides elements for formatting text with different meanings and appearances:

Example: Text Formatting
<p> This is <strong>important</strong> text. </p> <p> This is <em>emphasized</em> text. </p> <p> This is <b>bold</b> text. </p> <p> This is <i>italic</i> text. </p> <p> This is <mark>highlighted</mark> text. </p> <p> This is <code>code</code> text. </p>

Output:

This is important text.

This is emphasized text.

This is bold text.

This is italic text.

This is highlighted text.

This is code text.

Semantic HTML Elements

Semantic elements clearly describe their meaning to the browser and to developers. They improve accessibility and SEO:

Element Purpose
<header> Introductory content or navigation
<nav> Navigation links
<main> Main content of the page
<article> Self-contained content (blog post, news article)
<section> Thematic grouping of content
<aside> Content tangentially related to main content
<footer> Footer information (copyright, links)

Semantic Page Structure

Here's how to structure a complete web page using semantic elements:

Example: Semantic Page Layout
<!DOCTYPE html> <html> <body> <header> <h1>My Website</h1> <nav> <a href="../../index.html">Home</a> <a href="/about">About</a> </nav> </header> <main> <article> <h2>Blog Post Title</h2> <p>Blog content here...</p> </article> </main> <footer> <p>© 2026 My Website</p> </footer> </body> </html>

Output Structure:

<header>

My Website

Home | About
<main>

Blog Post Title

Blog content here...

<footer>

© 2026 My Website

Key Takeaways

  • HTML elements provide structure through tags
  • Use text formatting elements for meaning: <strong>, <em>, <mark>
  • Semantic elements improve accessibility and SEO
  • Use <header>, <nav>, <main>, <footer> for page structure
  • Semantic HTML makes code more readable and maintainable
